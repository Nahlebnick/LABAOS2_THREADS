# Laba 2 OS

## Условие задачи

**Задача:** Написать программу для консольного процесса, который состоит из трех потоков: `main`, `min_max` и `average`.

### 1. Поток Main (основной)
Выполняет следующие действия:
1. Создает массив целых чисел. Размерность и элементы массива вводятся пользователем с консоли.
2. Создает и запускает потоки `min_max` и `average`.
3. Ожидает завершения работы обоих потоков.
4. Заменяет максимальный и минимальный элементы массива на среднее значение элементов массива (полученное от потока `average`).
5. Выводит полученный массив на консоль.
6. Завершает работу программы.

### 2. Поток Min_Max
Выполняет следующие действия:
1. Ищет минимальный и максимальный элементы в массиве.
2. Выводит найденные значения на консоль.
3. **Имитация нагрузки:** После каждого сравнения элементов поток «засыпает» на **7 миллисекунд**.
4. Завершает свою работу.

### 3. Поток Average
Выполняет следующие действия:
1. Вычисляет среднее арифметическое значение элементов массива.
2. Выводит найденное значение на консоль.
3. **Имитация нагрузки:** После каждой операции суммирования элементов поток «засыпает» на **12 миллисекунд**.
4. Завершает свою работу.

## Технический стек

- Язык: C++ (Standard: C++17)
- Платформа: Windows (Windows API)
- Среда разработки: Visual Studion 2022

## Структура проекта

Решение состоит из трех проектов:
```text
Project Root
├── Libs/
│   ├── inputUtils/       # Утилита для ввода с консоли
│   ├── thread/            # Класс для работы с потоками
└── src/
│   ├── Main/       # Основной поток
│   ├── proc/       # Потоки для нахождения минимума/максимума и среднего арифметического
```

## Сборка и запуск

1. Откройте файл решения "LabOS2.sln" в Visual Studio.
2. Соберите решение (Build Solution / F7).
3. Запустите проект (Ctrl + F5).

Все исполняемые файлы (.exe) должны находиться в одной директории для корректной работы функции CreateProcess.
